------------------------------------COMPARA YA-----------------------------

Este es el primer prototipo de la plataforma compara ya. Una aplicación va a comparar el precio de un determinado producto (como jugos, botanas, artículos de casaste básica, etc) en disferentes tiendas de auto servicio, para que la gente pueda ver en que establecimiento le conviene más comprar un determinado producto.

Con esta aplicación, se pretenderá solucionar los problemas económicos de las familias mexicanas, e incentivará a las marcas y a las tiendas de autoservicio, a ser más competivivos con el precio de los productos que venden.
